# Where in the World
 
## Descripción

"Where in the World" es una aplicación web que permite visualizar información detallada sobre todos los países utilizando la API REST Countries. La aplicación fue creada siguiendo los diseños proporcionados y desarrollada completamente con HTML, CSS y JavaScript puro.

Permite al usuario:
- Visualizar todos los países en orden alfabético.
- Buscar países por nombre.
- Filtrar países por región.
- Ver detalles de un país en una página aparte.
- Navegar a los países colindantes desde la página de detalle.
- Cambiar entre tema claro y oscuro.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Iconos: Font Awesome
- API: REST Countries
- Fuente: Nunito Sans

## Funcionalidades destacadas

- **Búsqueda en tiempo real**: filtra países mientras el usuario escribe.
- **Filtros por región**: mantiene la selección al navegar entre páginas de detalle.
- **Empty State**: cuando no hay resultados, se muestra un mensaje para el usuario.
- **Responsive**: diseñado para desktop, laptop, tablet y móvil, siguiendo los puntos de quiebre recomendados.
- **Tema oscuro y claro**: cambia dinámicamente, con persistencia usando localStorage.

## Enfoque del desarrollo

La aplicación fue desarrollada desde cero, sin frameworks, siguiendo estos principios:
- **Maquetación manual**: uso de Flexbox y CSS Grid para organizar las tarjetas y la información.
- **Código limpio y modular**: separación de lógica de renderizado, filtros y búsqueda en funciones reutilizables
- **Experiencia de usuario**: búsqueda rápida, mensajes claros en estados vacíos y navegación fluida entre páginas.
- **Responsive y accesible**: todos los elementos se adaptan a diferentes tamaños de pantalla y mantienen una lectura clara.
